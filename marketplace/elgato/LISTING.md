# AgentDeck — Elgato Marketplace listing

## Version

`1.0.0.0` (product version `1.0.0`)

## Description

AgentDeck turns Stream Deck and Stream Deck + into a live control surface for AI coding agents. See Claude Code, Codex, OpenCode, and OpenClaw sessions at a glance, follow current tools and attention states, choose prompt options, steer supported sessions, stop work, and review usage without leaving your device.

The plugin is a thin client and requires the free AgentDeck macOS app or AgentDeck daemon running locally on port 9120. It does not collect analytics, embed the daemon, or modify shell configuration. A regular 15-key Stream Deck profile and a Stream Deck + profile are included.

## Release notes

First public release. Includes dynamic session keys, Stream Deck + dial and touch-strip actions, bundled profiles for Stream Deck and Stream Deck +, multi-agent state and attention rendering, prompt steering, stop and mode controls, and usage views.
